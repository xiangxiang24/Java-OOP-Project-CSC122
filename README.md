## Patient Management System

## 🗂️ Project Title
CSC122 Java Object Oriented Programming Project. This Project related to admin system for manage the patient information in hospital.

## 📦 Problem Statement
In current hospital management, staff often face several challenges such as being unable to 
access patient details quickly during emergency situations, missing personal information, 
and difficulty tracking the latest updated records.

To address these issues, this system was designed to provide everything in one place. 
It allows staff to view all patient information on a single page, check and automatically 
calculate total medical expenses, and monitor admitted patient status with clearly 
identified room numbers. The system also supports classifying patients by type and disease, 
displaying data in a readable list format, adding new patient records to file, and searching 
for any patient instantly by their ID making hospital data management more efficient and 
reliable.

This is just a basic system that is mostly found in many hospitals, but I believe that 
in the future, I will have a chance to come back to consider this process and improve it 
until it can be useful for public management systems.

## 🚀 Features
Feature 1: Display all patients with full details including ID, name, age, blood, gender, birthdate, contact, disease, type, room, block, admit days, and appointment date.

Feature 2: Add new patient data by selecting the patient type (Inpatient / Outpatient) and filling the required information following the system

Feature 3: Search and display specific patient's information by entering thier ID (5-digits)

Featuer 4: Filter and display patient data by searching from disease

Feature 5: Filter and display patient data by searching from patient building location (e.g. HB/FB)

Feature 6: Filter and display patient data by searching from patient type

Feature 7: Add bill for each patient by selecting from patient ID, including description (Name of medicine) and price

Feature 8: Check and display the full bill summary for a specific patient by patient ID, including basic expenses, extra charge and total expense
